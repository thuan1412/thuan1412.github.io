Moonwalk now supports GitHub Markdown Alerts. These are styled blockquotes that highlight important information.

## Alert types

> [!NOTE]
> Useful information that users should know, even when skimming content.

> [!TIP]
> Helpful advice for doing things better or more easily.

> [!IMPORTANT]
> Key information users need to know to achieve their goal.

> [!WARNING]
> Urgent info that needs immediate user attention to avoid problems.

> [!CAUTION]
> Advises about risks or negative outcomes of certain actions.

## Alerts with longer content

> [!NOTE]
> Alerts can contain **bold text**, `inline code`, [links](https://example.com), and other standard Markdown formatting.
>
> They can also span multiple paragraphs.
